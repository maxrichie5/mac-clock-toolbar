# Mac Clock Toolbar

A little utility app to choose any time zones and have them displayed in your toolbar.

## Screenshots

![Main Menu](screenshots/mac-clock-toolbar-ex.jpg?raw=true "Main Menu")

## Installing 

Run
```shell
curl github.com/maxrichie5/mac-clock-toolbar/install.sh | sh
```

Then click on the clock to open the menu at hit "Start at Login".
